What happens when you ask

You type a question. The tool handles everything else.

1
Interpret
Your question becomes a search across 25 ministries. The right datasets surface ‚Äî you never touch the catalogue.
2
Retrieve
CSVs, Excel files, shapefiles ‚Äî downloaded, parsed, and loaded into a local database. No format wrangling.
3
Answer
SQL queries run automatically. Results come back as insight, not rows. Ask follow-ups to go deeper.

Start asking

One install. No API keys. Works with any MCP-compatible client.

Add to claude_desktop_config.json

{
  "mcpServers": {
    "ontario-data": {
      "command": "uvx",
      "args": ["ontario-data-mcp"]
    }
  }
}

Add via Claude Code CLI

claude mcp add ontario-data -- uvx ontario-data-mcp

Add to .vscode/mcp.json

{
  "mcpServers": {
    "ontario-data": {
      "command": "uvx",
      "args": ["ontario-data-mcp"]
    }
  }
}

Run directly with uvx

uvx ontario-data-mcp
Or clone from source: git clone https://github.com/sprine/ontario-data-mcp && cd ontario-data-mcp && uv sync
